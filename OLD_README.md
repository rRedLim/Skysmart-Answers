### Установка

1. Подготовка к работе

   ```bash
   $ git clone https://github.com/xartd0/Skysmart-Answers.git
   $ pip install -r requirements.txt
   $ cd Skysmart-Answers
   ```

2. Небольшая настройка

   ```bash
   $ nano utils/config.py
   $ auth_creds = {
    'email': 'Почта',
    'password': 'Пароль'
   }
   ```
   
3. Запуск

   ```bash
   $ python template.py
   ```
